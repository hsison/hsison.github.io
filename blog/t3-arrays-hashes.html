Arrays and Hashes

Today were going to talk about Arrays and Hashes. We're going to discuss when and how to use them and also the differences between the two. Well, here goes nothing!

Arrays

An array is an ordered list of things. Array's are used to store collections of data and can be an ordered list of anything you would like such as: numbers, letters, strings, etc.
Here is a very quick lesson about counting in arrays.
The positions of the elements in an array always start from zero. The first element is considered the "zero" element and the second would be considered the first, and so on.

For example:
letters= [A, B, C, D, E, F]
Counting breakdown:
A would be the zero element,
B would be the 1st element,
C would be the 2nd element,
D would be the 3rd element,
E would be the 4th element,
F would be the 5th element.
Get it?

Creating new/empty Arrays:

You create an empty array by making a new array object and storing it in a variable.

Basic example:
array = Array.new
Usable Example:
letters = Array.new(15)

Array Literal:

An array literal is a list of variables wrapped in square brackets and separated by commas such as:
letters= [A, B, C, D, E, F].

Accessing individual variables:

To retrieve individual variables is very simple, just remember our array counting lesson.
(The first element is considered the "zero" element and the second would be considered the first, and so on)

letters= [A, B, C, D, E, F]
letters[2] (answer would be C)
letters[5] (answer would be F)
letters[2,4] (answer would be C and E)
letters[0..2] (answer would be A, B, C)

Hashes

Hashes are a lot like arrays because they are also used to store data. Instead of just creating an ordered list of elements (like an array), a hash is a collection of key/value pairs.
A hash variable can be created the same way as an array variable. The simplest way it to fill the hash object with the key/value pairs.

Creating a new hash:

grades = Hash.new
grades["Hannah"] = B
grades["Tyler"] = A
grades["Renee"] = C
grades["Rachel"] = B
puts grades["Hannah"]

Hash Literals:

Like arrays, hashes can be created with hash literals. Hash literals use the curly braces instead of square brackets. Like I said earlier hashes are a collection of key/value pairs.
You join the key/value pairs using =>(or the rocket as some people refer to it).

Example:
grades = {
"Hannah" => B,
"Tyler" => A,
"Renee" => C,
"Rachel" => B
}
puts grades["Hannah"]

Accessing Variable in the hash:

Just how you access individual variables in arrays, you can also in hashes. Because hashes are unordered, the order you loop over the key/value pairs with may not be the same you inserted them. Here we will use the .each

method:
Example:
grades = {
"Hannah" => B,
"Tyler" => A,
"Renee" => C,
"Rachel" => B
}
grades.each do|name, grade|
puts “#{name}: #{grade}”
end
